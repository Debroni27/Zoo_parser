
Функция get_categories_data_for_current_pet()

    Принимает в себя любое значения из списка следущих валидных аргументов:

        [
            "Товары и корма для собак",
            "Товары и корма для кошек",
            "Товары и корма для птиц",
            "Товары и корма для грызунов",
            "Товары и корма для рыб",
            "Товары и корма для рептилий",
            "Товары и корма для хорьков",
        ]

    Пример:

        get_categories_data_for_current_pet("Товары и корма для собак")

Функция get_urls_for_current_pet_products()

    Принимает в себя любое значения из списка следущих валидных аргументов:

        [
            "sobak",

            "gryzunov",
            "ryb",

        ]
